ABOUT

  This is a tiny plugin to auto-reload user themes inside Binary Ninja; it is
  designed to make the theme authoring process easier.

INSTALL

  Navigate to your user plugins folder, then simply clone the repo:

    $ git clone git@github.com:jonpalmisc/themewatch.git

USAGE

  1. Turn on your computer
  2. Carefully, open Binary Ninja
  3. Make a change to a theme in your user themes directory
  4. Watch in amazement as Binary Ninja refreshes the theme automatically
